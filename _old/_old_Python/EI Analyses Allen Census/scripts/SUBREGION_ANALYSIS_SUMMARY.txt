- Subregion-level (n=20): r = 0.313, p = 0.180 (weak, not significant)

WHY THE CORRELATION DROPPED:
The stronger lobe-level correlation was partially an artifact of aggregation.
Multiple subregions within the same lobe share the same rCMRGlc value (since
we only have lobe-level metabolic data), creating artificial clustering that
inflates correlation when averaged. The subregion analysis reveals the true
within-lobe heterogeneity in E:I ratios.

KEY INSIGHT:
To properly test this hypothesis, we need SUBREGION-SPECIFIC rCMRGlc data
(e.g., Brodmann area-level PET measurements), not lobe-level values.

KEY LIMITATIONS
================================================================================

1. SAME rCMRGlc FOR ALL SUBREGIONS IN A LOBE
   Problem: All frontal regions (M1C, A13, A25, A32, A44-A45, A46) share
   rCMRGlc=35.3, even though their E:I ratios vary from 1.48 to 3.05.

   This creates horizontal "bands" in the scatter plot, limiting our ability
   to detect true correlations.

2. WITHIN-LOBE E:I HETEROGENEITY
   Example: Frontal lobe E:I ranges from 1.48 (A32) to 3.05 (M1C) - a 2-fold
   difference! But they all get the same metabolic value.

3. NEED FOR FINER-GRAINED METABOLIC DATA
   To properly test the hypothesis, we need Brodmann area-level or gyrus-level
   PET measurements, not lobe-level averages.

CONCLUSION & RECOMMENDATIONS
================================================================================

MAIN FINDINGS:
1. Subregion-level correlation (r=0.31) is WEAKER than lobe-level (r=0.67)
2. Neither analysis reaches statistical significance
3. Primary visual cortex (V1C) provides strongest supporting evidence
4. Within-lobe E:I heterogeneity is substantial (e.g., frontal: 1.48-3.05)

INTERPRETATION:
The hypothesis that higher E:I ratios correlate with higher glucose metabolism
receives WEAK support from this analysis. The limitation is that we're forced
to assign the same lobe-level rCMRGlc to all subregions within that lobe,
masking true subregion-specific relationships.

RECOMMENDATIONS:
1. Obtain Brodmann area-level or gyrus-level PET metabolic data
2. Consider layer-specific E:I analysis (cortical layers 2/3 vs 5/6)
3. Investigate cell subtype composition (PV, SST, VIP interneurons)
4. Examine metabolic markers in single-cell RNA-seq data (e.g., glycolysis genes)
5. Focus on primary cortices (V1, M1, S1, A1) where both E:I and metabolism vary most
