WHAT WAS WRONG:
================================================================================

ORIGINAL (INCORRECT) MAPPING:
  V1C → "Primary_visual_cortex" (treated as separate lobe, rCMRGlc=40.1)
  V2, A19 → "Occipital_lobe" (rCMRGlc=35.8)

PROBLEM:
  Primary visual cortex (V1C, Brodmann area 17) IS anatomically part of the
  occipital lobe, not a separate lobe. While Heiss et al. may have reported
  V1 separately due to its higher metabolism, this doesn't justify treating
  it as an independent lobe in our analysis.

CORRECTED MAPPING:
  V1C, V2, A19 → "Occipital_lobe" (all use rCMRGlc=35.8)

================================================================================
WHY THE CORRELATION DISAPPEARED
================================================================================

The original "correlation" was driven ENTIRELY by treating V1C as a special
case:

BEFORE: V1C was an outlier point at:
  - E:I = 4.42 (highest)
  - rCMRGlc = 40.1 (highest)

  This single outlier point pulled the regression line upward, creating the
  appearance of a positive correlation.

AFTER: V1C is correctly grouped with occipital lobe at:
  - E:I = 4.42 (still highest among subregions)
  - rCMRGlc = 35.8 (same as other occipital regions and parietal lobe)

  Without the unique high metabolism value, V1C no longer acts as an outlier
  driving the correlation.

================================================================================
IMPLICATIONS FOR HYPOTHESIS
================================================================================

HYPOTHESIS: Higher E:I ratio correlates with higher glucose metabolism

RESULT WITH CORRECTED ANALYSIS: **HYPOTHESIS NOT SUPPORTED**

Evidence:
  ✗ No significant correlation at lobe level (r=0.127, p=0.838)
  ✗ No significant correlation at subregion level (r=0.073, p=0.761)
  ✗ Occipital lobe has highest E:I (3.18) but not highest metabolism
  ✗ Parietal and Occipital have same metabolism (35.8) but different E:I (2.35 vs 3.18)
  ✗ Regression explains <2% of variance

WHY THE ORIGINAL ANALYSIS SHOWED A CORRELATION:
  The apparent correlation was an ARTIFACT of:
  1. Incorrectly treating V1C as a separate "lobe"
  2. Assigning V1C a unique high rCMRGlc value (40.1)
  3. This created a single outlier point that drove the correlation

ACTUAL FINDING:
  With correct anatomical classification, there is NO relationship between
  E:I ratio and rCMRGlc in human cortex based on available data.

================================================================================
LIMITATIONS THAT REMAIN
================================================================================

1. LOBE-LEVEL rCMRGlc DATA
   - We assign same rCMRGlc to all subregions within a lobe
   - Masks potential subregion-specific relationships
   - V1C may truly have higher metabolism (40.1) than V2/A19, but we can't
     test this with current lobe-averaged data

2. SMALL SAMPLE SIZE AT LOBE LEVEL
   - Only n=5 lobes for correlation analysis
   - Low statistical power even if correlation exists

3. HEISS V1 DATA NOT USED
   - Heiss reported rCMRGlc=40.1 for primary visual cortex
   - We're using the broader occipital lobe value (35.8) instead
   - This may underestimate V1's true metabolism

4. MISSING SUBREGIONS
   - 14 of 34 cortical regions not mapped to Heiss lobes
   - Reduces sample size for subregion analysis

================================================================================
SCIENTIFIC LESSON
================================================================================

This correction demonstrates the importance of:

1. **Anatomical accuracy**: Always use correct anatomical classifications,
   even if data sources report regions separately

2. **Outlier influence**: A single misclassified data point (V1C with unique
   high metabolism) can create spurious correlations

3. **Data granularity**: Using lobe-level metabolic data with region-level
   cellular data limits our ability to detect true relationships

4. **Transparency**: When errors are found, correct them fully and document
   the impact on conclusions

THE BOTTOM LINE:
The hypothesis that E:I ratio correlates with glucose metabolism is NOT
supported by this analysis when V1C is correctly classified as part of
occipital lobe. The original correlation was an artifact.
