THE BIG PICTURE
================================================================================

ORIGINAL HYPOTHESIS: E:I ratio correlates with metabolism
RESULT: ✗ NOT SUPPORTED

NEW DISCOVERY: Functional classification explains metabolism
RESULT: ✓ SUPPORTED (p=0.0089)

WHY THIS MATTERS:
  - Sensorimotor cortices (visual, motor, somatosensory) process continuous
    external input → require sustained high activity → HIGH metabolism

  - Associative cortices (language, memory, emotion) perform integrative
    processing → more sparse activity → LOWER metabolism

  - E:I balance is tightly regulated (~3:1) across ALL cortex for stability
    → NOT a variable that changes with metabolic demand
